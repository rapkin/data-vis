## Installation

* Install Node.js
* `npm install -g bower gulp coffee-script`
* `npm install` - node dependencies
* `bower install` - frontend dependencies
* `pip install -r requirements.txt` - to install all dependencies

## Database

* Install PostgreSQL
* `createdb data-vis`
* `createuser postgres`

## Build

* `gulp` - build once
* `gulp dev` - build and watch (dev mode)
